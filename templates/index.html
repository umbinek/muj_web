Jak to celé funguje?

Tento web slouží jako hlavní centrum pro moje IoT zařízení. Celé to běží na Raspberry Pi 3, které funguje jako server a datové úložiště.

Hardware a senzory
  • Měření: Přímo na malině mám DHT senzor. Další data (třeba kvalitu vzduchu) posílá ESP32-C3 přes Wi-Fi.
  • Kamery: Mám tu ESP32-CAM, která fotí zahradu, a Raspberry Pi Zero 2, které přes software Motion hlídá pohyb.
  • Úložiště: Fotky a videa se ukládají na USB disk. Mám nastavený skript, který každý den smaže záznamy starší než 7 dní, aby se disk nezaplnil.
Vývoj a automatizace
  • Kódování z mobilu: Programuju v prohlížeči přes Code-server, který mi běží v Termuxu na telefonu Xiaomi Mi 9 SE.
  • GitHub Actions: Jakmile udělám git push, automaticky se spustí testy (Pytest), které zkontrolují, jestli jsem v kódu neudělal chybu.
  • Dostupnost: Díky Tailscale se k serveru připojím odkudkoliv. Web běží jako linuxová služba, takže se po restartu sám zapne.
